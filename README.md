# Klou Store - Professional POS System

Modern point-of-sale application built with React + TypeScript + Tailwind CSS.

## ✨ Features

✅ Home/Checkout - Manual entry + product search  
✅ Shopping Cart - Add, remove, edit quantities  
✅ Payment - 4 payment methods (Cash, Card, Whatnot, Gift)  
✅ Barcode Scanner - Manual code input, create products  
✅ Inventory - Full CRUD for products  
✅ Transactions - Daily sales history  

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 📦 Deploy to Railway

```bash
git add .
git commit -m "Klou Store POS"
git push
```

Railway will auto-detect and deploy!

## 📁 Project Structure

```
src/
  ├── components/     # Reusable components
  ├── screens/        # 6 main screens
  ├── store/          # Zustand state
  ├── types/          # TypeScript types
  ├── App.tsx
  └── main.tsx
```

## 🎨 Design

- **Colors**: White, Black, Blue (Square POS style)
- **Framework**: React 18 + Vite
- **State**: Zustand
- **Styling**: Tailwind CSS
- **Language**: TypeScript
